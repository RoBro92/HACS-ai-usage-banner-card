# Changelog

## 0.2.0

- Adds Lovelace visual editor support.
- Adds Gemini and Codex preset toggles that create rows with default logos and MQTT entities.
- Adds a local browser demo at `demo/index.html`.
- Adds a dependency-free Node collector runner with MQTT discovery publishing.
- Adds Linux/LXC, macOS, and Windows installer scripts.

## 0.1.0

- Initial release scaffold.
- Adds `custom:ai-usage-banner-card`.
- Supports multiple model rows with 5-hour and weekly allowance sensors.
- Adds MQTT discovery collector documentation and examples.
